# Gabriele Tausch – Website

## GitHub Pages einrichten

1. Neues Repository auf GitHub erstellen (z. B. `gabriele-tausch`)
2. Alle Dateien hochladen (index.html, css/, js/, images/)
3. Im Repository: **Settings → Pages → Branch: main → Save**
4. Die Seite ist dann erreichbar unter:  
   `https://DEIN-USERNAME.github.io/gabriele-tausch/`

## Eigene Domain (optional)
Falls du eine eigene Domain möchtest (z. B. gabriele-tausch.org):
- Im Repository eine Datei `CNAME` erstellen mit dem Inhalt: `gabriele-tausch.org`
- Beim Domain-Anbieter einen CNAME-Eintrag auf `DEIN-USERNAME.github.io` setzen
